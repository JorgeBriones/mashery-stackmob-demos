Mashery Network API Demo Apps in iOS + StackMob

We built some API powered mobile apps using Xcode, native framework components, and StackMob for backend persistence. These apps can help you understand how to make API calls to various Mashery Network APIs, parse results, integrate data with native Cocoa UI components and persist data in the cloud using StackMob backend services.

Sample apps are open source, of course.

View project on GitHub

Xcode (iOS) Demo Apps

Active.com

This demo app uses Active.com's Search API to find activities by location (postal code or city,state,country), plotting them on a map in a native map view. Activity data is stored on StackMob.

Edmunds.com

This demo app uses Edmunds.com Dealer API to find and plot automobile dealerships. Dealership location data is persisted on StackMob.

Coca-Cola Enterprises

This demo app uses Coca-Cola Enterprises Location API to locate CCE customer locations. Favorite locations are stored on StackMob.
